Package Manager

npm

npm is the default package manager for node and node comes bundled with npm. npm is used to manage the packages or dependencies. yarn is popular alternative to npm

package.json

The way npm works is with the package.json file, this file specifies what packages the project is using and its version. When you download a project all its dependencies aren't included instead you run the terminal command npm i to install the dependencies in the correct version for the project.

scripts

Scripts can also be defined in the package.json which will run with the npm run script-name Installing a package locally (only in this project) npm i package-name Installing a package globally (access in all projects) npm i -g package_name
