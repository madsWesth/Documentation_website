Javascript Concepts


Variables

Javascript is a dynamically typed language so you don't have to identify your variables with a type, the type will be inferred.
Variables can be declared with different keywords and what keyword you use controls some properties of the variable.

  • Variable with the let keyword is a variable you can reassign.
  • Variable with the const keyword is a variable you cannot reassign.
let and const are new to Javascript and were introduced in ES6. The reason for the introduction of these new keywords was that the older var keyword was unintuitive and caused a lot of errors. The new keywords are block scoped which is more similar to how variables behave in other programming languages.

Callback functions

A callback function is a function that is passed to another function. These are often used in asynchronous programming as a function to be called after an asynchronous operation finishes

Clean code / code conventions

  • Don't use the var keyword unless for support on old browsers
  • Use const instead of let wherever possible.
  • Don’t scope variables outside functions if avoidable.
  • Use functional programming style wherever possible usually with loop methods. This is a replacement to the imperative fori style
  • Be consistent with the code style strings with double quotes or single quotes? End a statement with ;? indentation? This doesnt really matter for the program but it makes code more readable if its all consistent. Companies will have their own style guides that should be followed for this reason. This also extends to conventions that are used. Some plugins can help with this, like ESLint.

Loops (loop/array methods) functional style coding

These are methods introduced in ES5 which allows you to perform common tasks on arrays very concisely in a very easy to read manner. These methods use a declarative approach which means you don’t explicitly specify how something should be done or in what order. These are some of the methods that i've used so far. 

    array.filter()
    array.forEach()
    array.find()
    array.findIndex()
    array.map()
    array.reduce()
